## Script clean memory swap and memory caches used 

### installation 

Follow this instructions :
``` 
$ git clone https://github.com/root-andry/clean-cache-swap
$ cd clean-cache-swap
$ chmod 755 ./clsswap 
$ chmod 755 ./install
$ sudo ./install
```

After your installation, open another terminal, you can check this command:
```
$ sudo clsswap
```
